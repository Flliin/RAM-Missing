# RAM-Missing-MICCAI2026

This repository will host the official implementation of **RAM-Missing: Retrieval-Augmented Missing-Aware Fusion for Robust Lung Cancer Subtyping**.

## Overview

RAM-Missing is a missing-aware multimodal fusion framework for robust lung cancer subtyping under severe modality missingness. The framework integrates whole-slide image features, CT imaging features, and RNA expression features, and introduces retrieval-augmented proxy generation and uncertainty-guided conditional routing to improve robustness when CT scans are unavailable.
